EJSApp Collab Session 1.0

This is the first release of the ejsapp_collab_session plugin.
This plugin lets you to create sessions for collaborative working with Java applets created 
with Easy Java Simulations (EJS) and added to your Moodle course by means of the ejsapp 
module.
The Java applets should be compiled with version 4.37 or older of EJS is required to properly
work.

This is a block plugin for Moodle so you should place the ejsapp folder in your /blocks 
folder, inside you Moodle installation.

This block needs the ejsapp module to be of any use.
You can find and download the ejsapp module at the plugins section in the Moodle.org webpage.

An explanation of EJSApp and EJSApp Collab Session is included in the folder "doc". You will
also find a txt file with relevant links.
